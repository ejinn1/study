# trim()

### <mark style="color:green;">trim()</mark>

> str.trim()
>
> 문자열.trim()

문자열의 양옆의 공백을 제거한 새로운 문자열을 반환한다

```javascript
const str = "   hello   "

str.trim() // "hello"
```

***

### <mark style="color:green;">trimStart()</mark>

문자열의 시작부분의 공백을 제거한 새로운 문자열을 반환한다

```javascript
const str = "   hello   "

str.trimStart() // "hello   "
```

### <mark style="color:green;">trimEnd()</mark>

문자열의 끝부분의 공백을 제거한 새로운 문자열을 반환한다

```javascript
const str = "   hello   "

str.trimEnd() // "   hello"
```
