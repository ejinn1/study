# lastIndexOf()

### <mark style="color:green;">lastIndexOf()</mark>

> str.lastIndexOf()
>
> (문자열 or 배열).lastIndexOf(문자)

```javascript
const str "abcda"

str.lastIndexOf('a') // 4

const arr = ['a','b','a']

arr.lastIndexOf('a') // 2
```

***

### <mark style="color:green;">indexOf()</mark>

> str.indexOf(s)
>
> (문자열 or 배열).indexOf(문자)

```javascript
const str = "abcd"

str.indexOf('b') // 1

const arr ['a', 'b']

arr.indexOf('b') // 1
```
